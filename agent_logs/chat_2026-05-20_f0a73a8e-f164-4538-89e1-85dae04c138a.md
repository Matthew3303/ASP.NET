## **Search Player** i **Preview Player** koriste **5 tipova Riot API zahtjeva**:

1. **Account API** (1x) - dohvata PUUID
   ```
   https://europe.api.riotgames.com/riot/account/v1/accounts/by-riot-id/{gameName}/{tagLine}
   ```

2. **Summoner API** (1x) - dohvata summoner level, profile icon
   ```
   https://euw1.api.riotgames.com/lol/summoner/v4/summoners/by-puuid/{puuid}
   ```

3. **League/Rank API** (1x) - dohvata rank, LP, wins/losses
   ```
   https://euw1.api.riotgames.com/lol/league/v4/entries/by-summoner/{summonerId}
   ```

4. **Match History API** (1x) - dohvata listu match IDs
   ```
   https://europe.api.riotgames.com/lol/match/v5/matches/by-puuid/{puuid}/ids
   ```

5. **Match Details API** (N x) - dohvata detalje svakog matcha
   ```
   {_baseUrl}/{matchId}
   ```

### **Razlika između Search i Preview**:
- **Search** (`saveToDb: true`): Sprema sve matcheve u bazu → potrebnih **do 24 API zahtjeva** (4 + do 20 match details)
- **Preview** (`saveToDb: false`): Ako je match već u bazi, koristi postojeće podatke → **4-24 zahtjeva** (ovisno koliko matcheva je novo)

Za 20 nedavnih matcheva: **~24 Riot API zahtjeva po pretrazi**.
